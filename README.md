PHP Coding Standards Fixer Ansible Role
=========

If you are a PHP developer and you use PHP Coding Standards Fixer.

Requirements
------------
  - minimum `php  5.6.0` 

Role Variables
--------------
None

Dependencies
------------
None

Example Playbook
----------------
    - hosts: php
      roles:
        - devsoul.php-cs-fixer

Check if it's installed:
------
    php-cs-fixer --version
